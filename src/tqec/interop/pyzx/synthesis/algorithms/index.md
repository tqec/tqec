This folder is for algorithms (plural, as in many approaches) used for synthesis.

Suggested structure for a given algorithm:

``` python
.  # this folder
|
├── ...  # files of other algorithms
|
├── name_of_algorithm.py: parent script
├── name_of_algorithm_utils.py
├── name_of_algorithm_utils_1.py
├── ...
├── name_of_algorithm_utils_n.py
|
├── ...  # files of other algorithms
|
```
